Instructions for playing Guessnum

  • You enter your name to start playing the game
  • The computer picks a three digit number with unique digits
  • Player tries to guess the computer's number
  • The computer remembers its number and the player's guesses
  • For each guess the computer will tell the player:

    How many right and how many in the right position. and the number of guesses so far

  • Eventually the computer's number is guessed and the results are stored in a database
  • For a given player the guessed number must be unique ! 




Guessnum is part of the Vicnum project which was developed for educational purposes to demonstrate common web vulnerabilities.
